Configuración Consultas

El sondeo de dispositivos se ejecuta en intervalos definidos por el temporizador de configuración. Cada consulta realiza un comando ping rápido a cada dispositivo para verificar su estado. El sistema mantiene un historial del estado de las consultas para cada dispositivo durante el número de días especificado en la configuración.
